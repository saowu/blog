pip3 常用命令

2020-04-03 · 2 min read
Python 日记

1、安装目录

~/Library/Caches/pip

2、查找需要安装的包

pip3 search <包名>

3、安装python包

pip3 install 
pip3 install <包名>==1.0.0
pip3 install -r requirements.txt 
pip3 install -i <源地址> <包名>
pip install -e <local project path> (这个命令相当于pip install develop）

4、源地址

阿里源：https://mirrors.aliyun.com/pypi/simple/ 
科大源：https://pypi.mirrors.ustc.edu.cn/simple/ 
清华源：https://pypi.tuna.tsinghua.edu.cn/simple 
官方源：https://pypi.python.org/

5、在配置文件中指定安装源

#直接在配置文件中添加源，如果没有配置文件可以手动添加一个 
/etc/pip.conf 
[global] 
timeout = 6000 
index-url = http://pypi.douban.com/simple

6、列出已经安装的包

pip3 list

7、列出过期的包

pip3 list -o

8、升级包

pip3 install <包名> -U
pip3 install -i <源地址> <包名> -U

9、安装的包所在目录

pip3 show -f <包名>

10、卸载安装

pip3 uninstall <包名> 
pip3 uninstall -r requirements.txt

11、生成requirements.txt文件

pip freeze > ./requirements.txt
